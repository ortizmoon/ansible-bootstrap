Роль деплоит Gitlab сервер на порту 8181.
Основные данные маунтятся в /var/opt/gitlab, на отдельный диск /deb/sdb.<br>
Бэкап настраивается через systemd таймер, выполняется в 3 часа ночи каждый день. <br>
Бэкапы кладутся через rclone, в Cloudflare R2. <br>
Восстановление бэкапа по скрипту в /usr/local/bin/gitlab-backup-r2.sh
